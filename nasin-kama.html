kama pi nasin Github

  1. o pana e wile Pull Request tawa ilo Github.

  2. o ante e jan.json, o pana e ni:

    { "name": "<nimi sina>", "url": "<linja pi lipu sina>" }

  3. o pana e lipu lili lon lipu sina!

kama pi nasin ante

sina ken ala kepeken ilo Github la sina ken kama lon nasin ante.
