kama pi nasin Github

  1. o pana e wile Pull Request tawa ilo Github.

  2. o ante e jan.json, o pana e ni:

    { "name": "<nimi sina>", "url": "<linja pi lipu sina>" }

  3. o pana e lipu lili lon lipu sina!

kama pi nasin ante

sina ken ala kepeken ilo Github la sina ken kama lon nasin ante.

o toki tawa kala pona Tonyu lon bucketfishy@gmail.com. o toki e nimi sina, e linja pi lipu sina.

pana pi lipu lili

sina lon sike la o pana e lipu lili tawa lipu sina. o pana e toki ni:

<iframe id="sike-pona" style="width: 100%; height: 3rem;" src="{{ '/jan/' | absolute_url }}nimi sina"></iframe>

o ante e nimi sina tawa nimi pi pana sina.

lukin la, ona li sama ni:

sina wile ala kepeken lipu lili la sina ken kepeken linja:

linja tawa kama: {{ '/jan/' | absolute_url }}nimi sina/next.html

linja tawa pini: {{ '/jan/' | absolute_url }}nimi sina/prev.html

linja tawa lipu ni: {{ '/' | absolute_url }}

o pona!
